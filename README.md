# Sống Đẹp — Gương Bác Trong Thời Đại Số

Dự án HCM202 — Khám phá triết lý sống đẹp của Hồ Chí Minh dành cho thế hệ Gen Z trong kỷ nguyên số.
https://hcm202-vyvy.vercel.app/

## Cấu Trúc Trang

| Trang | Mô tả |
|---|---|
| `index.html` | Trang chủ — hero với hiệu ứng particle, lộ trình 6 chương, danh sách tính năng |
| `triet-ly.html` | Triết lý đạo đức — 5 chuẩn mực Cần, Kiệm, Liêm, Chính, Chí công vô tư |
| `triet-ly-visual.html` | Triết lý đạo đức (bản visual) — bổ sung hình ảnh hero |
| `tam-guong-bac.html` | Tấm gương Bác Hồ — 5 câu chuyện lịch sử theo dạng bento grid |
| `ban-dang-song.html` | Trắc nghiệm tương tác — 10 tình huống thực tế đối chiếu chuẩn mực đạo đức |
| `thuc-tien-hom-nay.html` | Phân tích thực tiễn — tab tương tác, poll, Gimkit game, cam kết cá nhân |
| `toi-chon-cach-song.html` | Kết luận — bản đồ hành động Gen Z, tuyên ngôn thế hệ trẻ |
| `minh-bach-ai.html` | Báo cáo minh bạch — quy trình ứng dụng AI trong dự án |

## Tính Năng

- **Responsive** — tối ưu trên mobile, tablet, desktop
- **Dark mode** — hỗ trợ giao diện tối/sáng
- **Scroll reveal** — hiệu ứng fade-in khi cuộn trang
- **Scroll progress bar** — thanh tiến độ đọc ở đầu trang
- **Hero entrance** — hiệu ứng fade-in tuần tự cho tiêu đề
- **Bento grid layout** — bố cục dạng lưới linh hoạt
- **Canvas particles** — hiệu ứng hạt động trên trang chủ
- **Animated counters** — số liệu tự động đếm khi scroll tới
- **Interactive tabs** — chuyển tab nội dung (trang thực tiễn)
- **Quick poll** — bình chọn tương tác (trang thực tiễn)
- **Gimkit embed** — trò chơi củng cố kiến thức nhúng trực tiếp
- **Cam kết cá nhân** — widget nhập và lưu cam kết của người dùng

## Công Nghệ

- **Tailwind CSS** (CDN) — utility-first styling
- **Material Symbols** — icon system
- **Be Vietnam Pro + Lora** — font chữ
- **IntersectionObserver API** — scroll-triggered animations
- **Canvas API** — particle effects
- **Vanilla JS** — không phụ thuộc framework

## Triển Khai
